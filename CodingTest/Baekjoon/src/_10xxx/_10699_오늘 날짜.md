<h3>정답</h3>

<aside>

💡 채점 서버의 시간대(Timezone) : UTC+0로 설정하여 Date 가져오기

</aside>



```java
import java.time.LocalDate;
import java.time.ZoneId;

public class Main {
	public static void main(String[] args) {
		LocalDate now = LocalDate.now(ZoneId.of("UTC+0"));
		System.out.println(now);
	}
}

```
